# Portfolio Website

This is a simple portfolio website built with HTML, CSS, and JavaScript.

## Files

- `index.html` — main page
- `styles.css` — styles and responsive layout
- `script.js` — mobile menu toggle

## Run

Open `index.html` in a browser to view the site.

## Customize

1. Replace the name and text inside `index.html` with your own information.
2. Update the projects in the `#projects` section.
3. Change the contact links to your real profiles.
